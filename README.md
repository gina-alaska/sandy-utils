sandy-utils
===========

processing bits attached to the sandy (and other) data processing streams. 

basic structure:
* sandy-utils
  * bin  (actual heads of utils go here)
  * lib
    * sandy.rb (includes everything in sandy/)
    * sandy
      * version.rb 
      * other shared stuff
  * configs
    * foo.yml (config for util in bin, named foo)
